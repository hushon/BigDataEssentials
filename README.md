# Big Data Analysis Algorithms in PySpark
## Introduction

Big data analysis algorithms in Python and [Apache Spark](https://spark.apache.org/).  
The problems are from the homeworks in Stanford CS246 course.

## Requirements

 - Python 2.7
 - Apache Spark 3.6
